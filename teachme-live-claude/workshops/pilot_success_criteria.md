# Pilot Success Criteria

## Experience
- Realtime end-to-end voice turn latency p95 <= 3000ms.
- Conversation interruption success rate (barge-in) >= 95%.
- Session completion rate >= 90%.

## Reliability and Safety
- Structured response validity >= 98%.
- Image guardrail correctness >= 99% when no image uploaded.
- Low-confidence STT clarification behavior >= 95% correctness.
- Tool-loop failure rate <= 2%.

## Business and Adoption
- Tutor/learner CSAT >= 4.3/5.
- Weekly active users growth >= 15% during pilot.
- Cost per active session within agreed budget envelope.

## AI Quality
- Translation helpfulness (ko<->en) pass rate >= 90% on curated set.
- Tool precision >= 0.85 and recall >= 0.85 on eval harness.
- Hallucination critical incidents: zero tolerance in pilot.
